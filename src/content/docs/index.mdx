## What is 2D?

2D is a Layer-1 chain whose account model is USDC by default. It exposes both surfaces that real wallets use today:

- **Ethereum JSON-RPC** (`eth_*`) on `/` — for MetaMask, ethers.js, viem, etc.
- **Tron wallet HTTP API** (`/wallet/*`) — for TronLink, TronWeb.

Both surfaces operate on the **same underlying accounts**, so a transfer sent via TronLink is visible to `eth_getTransactionByHash` moments later.
