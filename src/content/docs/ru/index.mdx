## Что такое 2D?

2D — это L1-сеть, в которой USDC — базовый актив аккаунта по умолчанию. Наружу сеть отдаёт два внешних API, и оба уже из коробки понимают существующие кошельки:

- **Ethereum JSON-RPC** (`eth_*`) на `/` — для MetaMask, ethers.js, viem и прочих.
- **Tron wallet HTTP API** (`/wallet/*`) — для TronLink и TronWeb.

Оба API работают поверх **одного и того же состояния аккаунтов**. Перевод, отправленный из TronLink, сразу виден через `eth_getTransactionByHash`.
