## Что такое 2D?

2D — это L1-сеть с единым базовым активом (USD-stable) и двумя интерфейсами для кошельков, работающими поверх единого состояния аккаунтов:

- **Ethereum JSON-RPC** (`eth_*`) на `/` для MetaMask, ethers.js, viem.
- **Tron wallet HTTP API** (`/wallet/*`) для TronLink и TronWeb.

Перевод, отправленный из TronLink, сразу виден через `eth_getTransactionByHash`: оба API читают и пишут данные в одну и ту же таблицу аккаунтов. Блоки получают моментальную финальность (без необходимости ожидания подтверждений и риска реорганизации цепи), а независимые узлы-верификаторы (verifier) воспроизводят каждый блок до того, как передать обновленное состояние кошелькам.
