## Что такое 2D?

2D — это L1-сеть, в которой USDC — базовый актив аккаунта по умолчанию. Наружу она отдаёт две поверхности, к которым кошельки уже умеют ходить:

- **Ethereum JSON-RPC** (`eth_*`) на `/` — для MetaMask, ethers.js, viem и прочих.
- **Tron wallet HTTP API** (`/wallet/*`) — для TronLink и TronWeb.

Обе работают поверх **одного и того же набора аккаунтов**. Перевод, отправленный из TronLink, через долю секунды виден запросу `eth_getTransactionByHash`.
