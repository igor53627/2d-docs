## Что такое 2D?

2D — это блокчейн Layer-1, в котором USDC является базовой денежной единицей аккаунтов по умолчанию. Сеть предоставляет обе поверхности, которыми пользуются реальные кошельки сегодня:

- **Ethereum JSON-RPC** (`eth_*`) на `/` — для MetaMask, ethers.js, viem и т. п.
- **Tron wallet HTTP API** (`/wallet/*`) — для TronLink, TronWeb.

Обе поверхности работают над **одним и тем же набором аккаунтов**, поэтому перевод, отправленный через TronLink, моментально виден запросу `eth_getTransactionByHash`.
